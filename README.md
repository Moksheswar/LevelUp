# 🚀 LevelUp – Personal Routine Tracker

LevelUp is a modern productivity web application designed to help users build discipline and stay consistent with their daily routines. Whether it's working out, studying, reading, or building healthy habits, LevelUp provides a simple and intuitive way to track progress every day.

## ✨ Features

- ✅ Create and manage daily routines
- 📅 Track habit completion with streaks
- 📊 View progress and consistency
- 🎯 Set personal goals
- 💾 Local data persistence (your progress remains after page refresh)
- 📱 Fully responsive design
- ⚡ Fast and lightweight
- 🌙 Clean and modern UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
- Git & GitHub
- Netlify (Deployment)

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/Moksheswar/LevelUp.git
```

Navigate into the project

```bash
cd LevelUp
```

Open

```
index.html
```

## 📖 Usage

1. Open the application.
2. Add your daily habits or routines.
3. Mark tasks as completed.
4. Track your daily consistency.
5. Stay motivated and level up every day.
